# 📊 Customer Risk & Sales Dashboard

Aplicación web desarrollada con Python para el análisis de clientes y ventas, integrando base de datos SQLite y visualización interactiva con Streamlit.

---

## 🚀 Demo en vivo

👉 https://TU-APP.streamlit.app

---

## 🧠 Funcionalidades

- 📊 Análisis de ventas por cliente
- 🧾 Integración con base de datos SQLite
- 📈 Visualización de datos en tiempo real
- ⚡ Dashboard interactivo con Streamlit
-   Integración de datos desde CSV
-   Base de datos relacional (SQLite)
-   Análisis con SQL (JOIN)
-   Identificación de clientes riesgosos
-   Generación de reportes automáticos

---

## 🛠️ Tecnologías utilizadas

- Python
- Streamlit
- Pandas
- SQLite

## ▶️ Cómo ejecutar el proyecto

1. Clonar repositorio:

   git clone
   https://github.com/helio-developer/customer-risk-sales-system.git

2. Instalar dependencias:

   pip install -r requirements.txt

3. Ejecutar la app:

   streamlit run src/app.py⁠
