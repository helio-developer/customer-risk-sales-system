# 📊 Customer Risk & Sales System

Sistema de análisis de clientes y ventas desarrollado en Python y SQLite.

---

## 📁 Estructura del proyecto

customer-risk-sales-system/ │ ├── data/ │   ├── clientes.csv │   └── ventas.csv │ ├── src/ │   ├── database.py │   └── analysis.py │ ├── reports/ │   └── resumen.txt

---

## 🚀 Funcionalidades

- Integración de datos desde CSV
- Base de datos relacional (SQLite)
- Análisis con SQL (JOIN)
- Identificación de clientes riesgosos
- Generación de reportes automáticos

---

## 🛠️ Tecnologías

- Python
- SQLite
- SQL
